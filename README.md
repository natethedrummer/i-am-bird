# I Am Bird

A 3D physics-based game built in [Godot Engine](https://godotengine.org/) where you play as a bird navigating the world using realistic flight mechanics.

## Concept

You are a bird. Flap, soar, and glide through open environments using physics-driven flight. The game focuses on the tactile joy of movement — mastering thermals, wind, and momentum to travel with grace or reckless speed.

## Planned Features

- Physics-based flight with lift, drag, and weight simulation
- Open 3D environments to explore
- Collectibles, challenges, or objectives tied to movement skill
- Bird character with animated wings driven by flight state

## Project Structure

```
i-am-bird/
├── assets/
│   ├── models/       # 3D meshes (.glb, .obj)
│   ├── textures/     # Image assets
│   └── audio/        # Sound effects and music
├── scenes/           # Godot .tscn scene files
└── scripts/          # GDScript (.gd) files
```

## Getting Started

1. Install [Godot 4](https://godotengine.org/download)
2. Clone this repository
3. Open Godot and import the project folder
4. Run the main scene
